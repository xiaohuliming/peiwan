# 无畏契约武器叙事用途参考（中文转述版）

生成目的：给剧情游戏、二创乙游、动作分镜、角色武器偏好和 LLM 场面生成使用。  
资料基于 Valorant Wiki / Fandom 的武器页、信息框、Summary 与基础参数整理，并参考 BWIKI 的中文武器名。以下内容是中文转述与叙事化归纳，不是逐字翻译。

- Fandom 总入口：https://valorant.fandom.com/wiki/VALORANT_Wiki
- 武器分类参考：https://valorant.fandom.com/wiki/Category:Weapons
- 中文名参考：https://wiki.biligame.com/valorant/分类:英雄

## 使用方式

- 不要只把武器当伤害数字。剧情里武器更像“角色如何处理危险”的符号：是冷静、压迫、冒险、隐蔽、保护，还是失控。
- Phantom、Ghost、Spectre 适合潜入、烟雾、夜战和克制型角色。
- Vandal、Sheriff、Guardian 适合精准、骄傲、单发决断和强烈个人风格。
- Operator、Outlaw、Marshal 适合保护撤离、远距离压迫、狙击手心理战。
- Ares、Odin 适合火力压制、守点、掩护队友撤离和“别过来”的场面。
- Shorty、Bucky、Judge 适合伏击、狭窄空间、突然贴脸和角色被逼到极端。

## 佩枪

### 标配 / Classic

- 来源：https://valorant.fandom.com/wiki/Classic
- 基础定位：免费佩枪，半自动，12 发弹匣，低穿透；副模式近距离三连发。
- 叙事用途：最适合“新人、失去装备、最后手段”。它不豪华，但说明角色还能保持基本战斗力。
- 场面气质：临时、朴素、可靠、低资源。角色拿它打赢强敌，会突出冷静和临场判断。
- 适合角色：新成员、后勤被迫上场、任务失败后只剩最低配置的人。
- 写作例子：训练场第一课、弹药耗尽后的最后反击、女主第一次真正开枪。

### 短炮 / Shorty

- 来源：https://valorant.fandom.com/wiki/Shorty
- 基础定位：低价双发短管霰弹佩枪，近距离爆发强，远距离几乎无用。
- 叙事用途：伏击武器。它不是正面对枪，而是藏在门后、烟里、转角后的一瞬间。
- 场面气质：阴险、搏命、贴脸、肾上腺素。两发之后就要承担空窗，适合制造极限张力。
- 适合角色：擅长潜伏的人、被追杀到死角的人、表面温和但能下狠手的人。
- 写作例子：狭窄楼梯间反杀、假装无武装后突然近身、保护伤员时守住一扇门。

### 狂怒 / Frenzy

- 来源：https://valorant.fandom.com/wiki/Frenzy
- 基础定位：自动手枪，450 点，射速高，低穿透，小弹匣。
- 叙事用途：短距离混战和冲动进攻。它像“我没有时间瞄准，但我一定要冲过去”。
- 场面气质：急促、暴躁、巷战、贴近目标。子弹很快打空，适合写失控和风险。
- 适合角色：Phoenix、Raze、Neon 这类高速或高能角色，也适合没有经验但被迫冲锋的人。
- 写作例子：夜市追逐、车辆旁突发枪战、角色情绪失控后连续扣扳机。

### 鬼魅 / Ghost

- 来源：https://valorant.fandom.com/wiki/Ghost
- 基础定位：带消音的半自动佩枪，500 点，13 发弹匣，中穿透。
- 叙事用途：潜入、侦察、室内清点、低声解决威胁。它是“别让任何人知道我来过”的枪。
- 场面气质：冷静、隐秘、优雅、克制。比 Sheriff 更温柔，比 Classic 更专业。
- 适合角色：Cypher、Omen、Fade、Iso、Chamber，也适合执行暗线任务的女主。
- 写作例子：烟雾里一声轻响、监控室无声清场、角色把枪递给女主让她自保。

### 追猎 / Bandit

- 来源：https://valorant.fandom.com/wiki/Bandit
- 基础定位：600 点半自动佩枪，8 发弹匣，中穿透；近中距离精准爆头威胁高。
- 叙事用途：介于 Ghost 和 Sheriff 之间，适合写“轻装但讲究准度”的角色。它比 Ghost 更硬，比 Sheriff 更灵活。
- 场面气质：利落、老练、街头决斗、近中距离压迫。
- 适合角色：赏金猎人、便衣特工、旧小队成员、擅长单点解决问题的人。
- 写作例子：酒吧后巷对峙、仓库内快速拔枪、角色用一发精准射击打断敌人行动。

### 正义 / Sheriff

- 来源：https://valorant.fandom.com/wiki/Sheriff
- 基础定位：800 点左轮式高威力佩枪，6 发弹匣，高穿透，射速低、后坐强。
- 叙事用途：尊严、压迫和赌命。它适合“我只有几发，但每一发都算数”的场景。
- 场面气质：决斗、孤注一掷、高手证明、强烈个人风格。
- 适合角色：Chamber、Iso、Veto、Brimstone、Reyna；也适合写女主第一次真正镇住场面。
- 写作例子：空旷大厅里只剩一把 Sheriff、审讯室桌面上的威慑、雨夜一发打碎控制装置。

## 冲锋枪

### 蜂刺 / Stinger

- 来源：https://valorant.fandom.com/wiki/Stinger
- 基础定位：低价 SMG，1100 点，20 发弹匣，射速极高，低穿透；副模式为点射/短连发。
- 叙事用途：轻装突袭、预算不足、快速贴近。它像一把“把机会抢出来”的武器。
- 场面气质：短促、凶、近距离爆发、容易失控。
- 适合角色：Neon、Yoru、Raze、Phoenix，或者任何擅长制造近距离优势的角色。
- 写作例子：经济紧张的一回合、地下通道突入、角色靠速度抢下敌人长枪。

### 骇灵 / Spectre

- 来源：https://valorant.fandom.com/wiki/Spectre
- 基础定位：1600 点 SMG，30 发弹匣，带消音，适合短中距离和移动射击。
- 叙事用途：潜入和持续推进。它不像 Stinger 那么赌，也不像步枪那么张扬，是“专业但不高调”的选择。
- 场面气质：夜战、烟雾、走廊推进、稳定清场。
- 适合角色：Cypher、Fade、Omen、Gekko、Skye，也适合女主作为支援型角色的默认武器。
- 写作例子：烟中掩护撤离、关灯后的走廊清点、两人背靠背缓慢推进。

## 霰弹枪

### 雄鹿 / Bucky

- 来源：https://valorant.fandom.com/wiki/Bucky
- 基础定位：850 点泵动霰弹枪，5 发弹仓，低穿透；有近距离主射和中近距离副射。
- 叙事用途：守窄门、穷途反击、旧式粗暴武器。它适合写“我不需要优雅，只要你别进来”。
- 场面气质：沉重、等待、门后呼吸声、一次失误就会被反打。
- 适合角色：Breach、Deadlock、Brimstone，或被逼入仓库/地下室的角色。
- 写作例子：守住避难所入口、暴风雪港口集装箱伏击、角色单手压弹等待下一波敌人。

### 判官 / Judge

- 来源：https://valorant.fandom.com/wiki/Judge
- 基础定位：1850 点自动霰弹枪，5 发弹匣，低穿透，近距离连续火力强。
- 叙事用途：清狭小空间、制造恐惧、把敌人从门口打回去。它比 Bucky 更主动、更凶。
- 场面气质：暴力、密闭、连续爆响、压迫走廊。
- 适合角色：Raze、Breach、Reyna、Viper，或者任何愿意把战斗拖进近距离的人。
- 写作例子：实验室转角连杀、敌人破门瞬间反扑、角色为了保护女主站在门内侧。

## 步枪

### 獠犬 / Bulldog

- 来源：https://valorant.fandom.com/wiki/Bulldog
- 基础定位：2050 点步枪，24 发弹匣，中穿透；主射自动，副模式为较精准的三连发。
- 叙事用途：资源不够但仍要执行任务时的务实选择。它适合写“不是最强，但够稳”。
- 场面气质：实用、过渡、纪律性、团队经济紧张。
- 适合角色：Brimstone、Harbor、Skye、Gekko 这类重视团队节奏的人。
- 写作例子：任务中途补给不足、二线小队装备、角色把好枪让给别人自己拿 Bulldog。

### 戍卫 / Guardian

- 来源：https://valorant.fandom.com/wiki/Guardian
- 基础定位：2250 点半自动精确步枪，12 发弹匣，高穿透，远距离表现优秀。
- 叙事用途：耐心、纪律、单发审判。它适合写给“不急着开第二枪”的人。
- 场面气质：冷静、长廊、墙后威慑、精准爆头。
- 适合角色：Sova、Chamber、Iso、Sage、Veto。也适合写女主从慌乱变得冷静的成长。
- 写作例子：寺院长廊守线、穿墙打掉控制器、狙击枪不可用时的精确替代。

### 幻影 / Phantom

- 来源：https://valorant.fandom.com/wiki/Phantom
- 基础定位：2900 点自动步枪，30 发弹匣，中穿透，带消音；短中距离和烟雾作战强。
- 叙事用途：潜入、烟中近战、稳定控场。它适合专业特工，也适合“保护不是炫技”的人。
- 场面气质：克制、稳定、暗处连续射击、烟雾里看不清弹道。
- 适合角色：Omen、Viper、Cypher、Fade、Skye、Brimstone。
- 写作例子：烟雾封锁后近距离交火、夜间潜入不暴露位置、角色用短点射掩护女主撤退。

### 狂徒 / Vandal

- 来源：https://valorant.fandom.com/wiki/Vandal
- 基础定位：2900 点自动步枪，25 发弹匣，中穿透；远距离无明显伤害衰减，奖励精准爆头。
- 叙事用途：骄傲、决断、高手对枪。它适合“我只需要一个机会”的角色。
- 场面气质：锋利、强势、远距离点射、公开压制。
- 适合角色：Jett、Reyna、Iso、Yoru、Phoenix、Chamber。
- 写作例子：屋顶对枪、远距离一发逆转、角色把 Vandal 校准后交给主角表示认可。

## 狙击枪

### 飞将 / Marshal

- 来源：https://valorant.fandom.com/wiki/Marshal
- 基础定位：950 点轻型狙击枪，5 发，单倍镜，移动和换位较灵活。
- 叙事用途：侦察手、轻装狙击、早期任务压制。它不像 Operator 那么沉重，更适合快速转移。
- 场面气质：呼吸、角度、警告性第一枪、雪地或屋顶观察。
- 适合角色：Sova、Skye、Deadlock、Yoru。
- 写作例子：远处打掉探照灯、守住撤离路线、角色教女主“先听风再开枪”。

### 莽侠 / Outlaw

- 来源：https://valorant.fandom.com/wiki/Outlaw
- 基础定位：2400 点双管狙击枪，2 发弹仓，高穿透，中距离高威胁；双发后换弹压力明显。
- 叙事用途：高风险高亮点。它适合写“我有两次机会，第二次以后就轮到你保护我”。
- 场面气质：紧张、戏剧性、双发节奏、换弹危机。
- 适合角色：Chamber、Iso、Veto、Waylay，或喜欢赌博式行动的角色。
- 写作例子：两发分别打掉两个追兵、第二发救下女主后陷入换弹空窗、双人配合补位。

### 冥驹 / Operator

- 来源：https://valorant.fandom.com/wiki/Operator
- 基础定位：4700 点重型狙击枪，5 发，高穿透，双倍镜，移动慢但单发压迫极强。
- 叙事用途：区域封锁、保护撤离、心理压迫。它不只是杀伤，更是告诉敌人“这条路不能走”。
- 场面气质：寂静、长距离、等待、一声枪响后全场停顿。
- 适合角色：Chamber、Jett、Sova、Deadlock、Veto。
- 写作例子：高塔守撤离点、风雪中远程保护、敌人迟迟不敢穿过广场。

## 机枪

### 战神 / Ares

- 来源：https://valorant.fandom.com/wiki/Ares
- 基础定位：1600 点机枪，50 发弹匣，高穿透，持续射击时稳定性增强。
- 叙事用途：压制火力和守点。它适合“不让敌人靠近”的保护型场面。
- 场面气质：沉重、持续、火线封锁、墙后威胁。
- 适合角色：Brimstone、Breach、Deadlock、Harbor。
- 写作例子：守住走廊让队友撤离、穿墙逼退敌人、角色用火力给女主争取破解时间。

### 奥丁 / Odin

- 来源：https://valorant.fandom.com/wiki/Odin
- 基础定位：3200 点重机枪，100 发弹匣，高穿透，副模式可立即进入最高射速。
- 叙事用途：重火力、最终防线、压倒性场面。它是“我一个人把这里变成战线”的武器。
- 场面气质：轰鸣、震动、阵地战、压迫到近乎不讲理。
- 适合角色：Brimstone、Breach、Viper、Deadlock，也适合反派或大型防御设施守卫。
- 写作例子：基地被围攻时的最后防线、敌人被火力钉在墙后、角色扛着重机枪却仍不退。

## 近战与特殊武器

### 近战武器 / Melee

- 来源：https://valorant.fandom.com/wiki/Melee
- 基础定位：基础近战武器，不可丢弃；持有时移动速度快，背刺有额外威胁。
- 叙事用途：沉默、亲密距离、弹尽粮绝、非枪械解决。它比枪更贴身，也更私人。
- 场面气质：无声、危险、靠近、身体距离拉满。
- 适合角色：Yoru、Iso、Jett、Fade、Viper，也适合潜入剧情里的主角。
- 写作例子：无声割开束缚、背后制服守卫、角色把刀柄递给女主让她切断绳索。

### 黄金枪 / Golden Gun

- 来源：https://valorant.fandom.com/wiki/Golden_Gun
- 基础定位：特殊模式武器，1 发弹匣，高穿透，移动速度快，象征“一发定胜负”。
- 叙事用途：仪式化决斗、赌局、特殊训练、虚拟模拟。它不适合常规现实剧情，但很适合梦境、训练场、娱乐模式或反派游戏。
- 场面气质：夸张、戏剧、赌命、全场注视。
- 适合角色：Chamber、Phoenix、Yoru、Reyna。
- 写作例子：训练模拟里的最终一枪、反派设下的竞技规则、角色把胜负押在一次扣扳机上。

### 雪球枪 / Snowball Launcher

- 来源：https://valorant.fandom.com/wiki/Snowball_Launcher
- 基础定位：特殊模式武器，1 发装填，无限备用弹，偏节日/娱乐规则。
- 叙事用途：轻松章节、节日活动、队内破冰。它能把紧绷的战术组织写出生活感。
- 场面气质：玩闹、节庆、短暂和平、反差萌。
- 适合角色：Gekko、Raze、Phoenix、Neon、Clove、Miks。
- 写作例子：基地冬日活动、训练失败后的惩罚赛、冷面角色被迫参与小游戏。

## 角色配枪速查

- 潜入型：Ghost、Spectre、Phantom、Melee。
- 精准型：Sheriff、Guardian、Vandal、Marshal、Operator。
- 保护型：Phantom、Ares、Odin、Guardian、Operator。
- 冲锋型：Frenzy、Stinger、Spectre、Judge、Vandal。
- 伏击型：Shorty、Bucky、Judge、Ghost。
- 训练/新手型：Classic、Bulldog、Marshal。
- 高压恋爱桥段：Operator 远程保护、Phantom 烟中牵引、Shorty 门后共守、Guardian 长廊并肩、Melee 近距离解困。
