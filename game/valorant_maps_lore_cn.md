# 无畏契约地图剧情与场景氛围参考（中文转述版）

生成目的：给剧情游戏、二创乙游、LLM 场景生成和章节策划使用。  
资料基于 Valorant Wiki / Fandom 的地图页、信息框、Unique Features 与 Lore 段落整理，并参考 BWIKI 的中文地图名。以下内容是中文转述与剧情化归纳，不是逐字翻译。

- Fandom 总入口：https://valorant.fandom.com/wiki/VALORANT_Wiki
- 地图分类参考：https://valorant.fandom.com/wiki/Category:Maps
- 中文名参考：https://wiki.biligame.com/valorant/分类:英雄

## 使用方式

- 写主线：优先用 Ascent、Fracture、Pearl、Lotus、Abyss、Corrode，它们和跨世界、源晶、研究设施、古代遗迹、组织冲突联系更强。
- 写关系线：优先用 Haven、Bind、Icebox、Split、Sunset，它们有城市、故乡、企业入侵、秘密设施、旧伤与身份误会。
- 写日常和训练：Range 最适合；Breeze 可写假期伪装、岛屿潜入；Pearl 可写水下城市约会与危机并存。
- 写压迫感：Abyss 的坠落边界、Operator 架点、断崖追逐非常强；Fracture 的四象限夹击适合背叛和包围；Icebox 的寒冷、集装箱和古船适合孤立恐怖。

## 地图详表

### 亚海悬城 / Ascent

- 来源：https://valorant.fandom.com/wiki/Ascent
- 地点：Alpha Earth，意大利威尼斯 San Marco。
- 关键机制：两处 Spike 点都有可开关机械门；部分墙面和玻璃可被破坏，破坏后会改变本回合通路与视线。
- 剧情背景：一次来自 Omega 的 Spike 事件在威尼斯造成灾难，城市一大块区域被撕离地面，永久悬浮在空中。这是 Alpha 已知最严重的源晶灾难之一，也和捷风被误认有关。
- 场景氛围：破碎城市、悬空广场、钟楼与教堂影子、灾后封锁区、公众恐慌和官方掩盖。
- 适合剧情：同位体误会、城市灾难调查、被嫁祸的追逃、主角第一次接触跨世界真相。
- 乙游钩子：可以写成“在悬空废墟中查明他/她不是罪人”的章节。高处风声、坠落边缘、碎裂广场很适合关系升温和保护桥段。

### 源工重镇 / Bind

- 来源：https://valorant.fandom.com/wiki/Bind
- 地点：Alpha Earth，摩洛哥 Rabat。
- 关键机制：没有中路，取而代之的是两个单向传送器；传送会产生全图可感知的音效，传送出口有自动门。
- 剧情背景：Kingdom Corporation 在 Rabat 设立源晶炼制设施，并计划继续扩张，拆除阻碍项目的文化资产。当地居民已经看穿企业承诺背后的代价，对 Kingdom 的活动强烈反抗。
- 场景氛围：沙色城市、炼制工厂、旧城文化与工业设施冲突、传送门误导、抵抗组织暗线。
- 适合剧情：民众抗议、企业压迫、秘密运输、传送器误入、Cypher 故乡线。
- 乙游钩子：适合写“本地向导/情报贩子带你穿过旧城和厂区”。传送器可用于误会、错位会合、追逐中突然分别。

### 隐世修所 / Haven

- 来源：https://valorant.fandom.com/wiki/Haven
- 地点：Alpha Earth，不丹 Thimphu。
- 关键机制：三处 Spike 点；部分门墙有可破坏面板，破坏后可穿透或改变防守方式。
- 剧情背景：不丹首都的寺院受到源晶事件影响，地面出现巨大柱状异变，Kingdom 也进入现场，把寺院变成源晶箱储存点。
- 场景氛围：寺院、火焰、断壁、山城、宗教宁静被现代企业和源晶灾变撕裂。
- 适合剧情：祈祷与战斗并置、保护圣地、精神试炼、角色面对自己信念。
- 乙游钩子：适合慢热、疗愈、承诺类章节。三点结构可写成“三路同时行动”，让角色被迫分开又在寺院深处重逢。

### 霓虹町 / Split

- 来源：https://valorant.fandom.com/wiki/Split
- 地点：Alpha Earth，日本东京 Shinjuku。
- 关键机制：多组垂直绳索，强调上下层移动和高低差。
- 剧情背景：东京新宿形成企业开发与本地居民抵抗的明显分裂。Kingdom 办公楼和商业广场扩张到城市中，而居民试图夺回被挤压的生活空间和文化资产。
- 场景氛围：霓虹灯、雨夜巷道、高楼办公区、旧街区、企业巨型总部压在城市天际线上。
- 适合剧情：都市潜入、反企业行动、夜间追逐、双面身份、Yoru 日本线。
- 乙游钩子：非常适合现代都市恋爱感。可以让角色白天在企业区伪装，夜里穿过旧街区交换情报。

### 森寒冬港 / Icebox

- 来源：https://valorant.fandom.com/wiki/Icebox
- 地点：Alpha Earth，俄罗斯 Sakha 的 Bennett Island。
- 关键机制：横向滑索、高低差、双点都存在复杂掩体和可攀爬结构。
- 剧情背景：Kingdom 在北极群岛建立集装箱港口与研究设施，挖掘一艘被冰封的古代战船。研究发现这艘船和船员可能远比表面更神秘，吸引其他势力注意。地图中也有与 Yoru 面具相关的武士铠甲线索。
- 场景氛围：极寒、集装箱、科研港口、冰封古船、无人设施、破碎玻璃和秘密实验室。
- 适合剧情：极地封闭任务、古代遗物调查、失踪队伍、Yoru 祖先线、暴风雪中互相依靠。
- 乙游钩子：适合写“低温求生”和“一间临时避寒室”。角色平时冷淡，在极端环境下不得不靠近。

### 微风岛屿 / Breeze

- 来源：https://valorant.fandom.com/wiki/Breeze
- 地点：Alpha Earth，大西洋百慕大三角。
- 关键机制：A 点附近有绳索与自动门；地图空间感开阔。
- 剧情背景：神秘热带岛屿地下有源晶矿，岸边有搁浅货船，也存在功能异常的传送装置。这里对 Alpha 与 Omega 双方都有战略意义，并和 KAY/O 从未来时间线抵达当下的线索有关。
- 场景氛围：阳光海岛、遗弃船体、矿井、地下机房、热带假期表象下的军事价值。
- 适合剧情：假期伪装、岛屿潜入、传送异常、未来来客、海岸撤离。
- 乙游钩子：表面是海岛任务，实际是秘密矿区和传送门事故。很适合把轻松日常突然切到危机。

### 裂变峡谷 / Fracture

- 来源：https://valorant.fandom.com/wiki/Fracture
- 地点：Alpha Earth，美国 New Mexico 的 Santa Fe County。
- 关键机制：地图被切成四个中立象限，攻击方可通过跨地图单向滑索从两侧展开；有自动门、绳索和四个能量球。
- 剧情背景：Everett-Linde Research Facility 是 Alpha 与 Omega 两边 Kingdom 组织合作的绝密设施，目标是建造大型源晶对撞机。但启动当天设施被两名狙击手摧毁，爆炸造成大量人员死亡，留下失败合作的废墟。
- 场景氛围：峡谷实验室、对撞机残骸、黄沙、科研事故、双世界合作破裂。
- 适合剧情：科研阴谋、背叛、双狙击手、事故调查、Chamber 相关线。
- 乙游钩子：适合写“真相不只一边”。女主可以同时拿到 Alpha 与 Omega 两套记录，被迫判断谁在说谎。

### 深海明珠 / Pearl

- 来源：https://valorant.fandom.com/wiki/Pearl
- 地点：Omega Earth，葡萄牙 Lisbon。
- 关键机制：没有特殊机关，更强调清晰街区、长线控制和城市空间推进。
- 剧情背景：Omega Earth 气候濒临崩溃，Lisbon 面临被海潮吞没。VALORANT Legion 与 Kingdom Industries 合作建造地理穹顶，让部分城市和居民能在海面下继续生活。
- 场景氛围：水下城市、穹顶、海压、日常市井、灾后文明继续运转。
- 适合剧情：Omega 社会观察、敌方世界的人情味、潜水舱、海底约会、气候危机。
- 乙游钩子：非常适合让女主第一次意识到 Omega 并非单纯敌人。水下穹顶的美丽和脆弱可以承载很强的暧昧与道德冲突。

### 莲华古城 / Lotus

- 来源：https://valorant.fandom.com/wiki/Lotus
- 地点：Omega Earth，印度 Karnataka 的 Western Ghats。
- 关键机制：三处 Spike 点；旋转门、可破坏墙、绳索和静音下落。
- 剧情背景：Omega 印度的西高止山脉发现了古代失落城市“City of Flowers”。它由古老 Guardians 建造，是多元宇宙节点。VALORANT Legion 激活了城市，Alpha Protocol 担心 Alpha 对应遗址已成废墟，激活可能波及数百万人。
- 场景氛围：雨林古城、石门、花纹机关、旋转密道、多元宇宙神庙。
- 适合剧情：遗迹探索、古代文明、守护者器物、Astra/Harbor 线、多元宇宙节点危机。
- 乙游钩子：适合写“机关迷宫中的双人解谜”。旋转门可以制造错过、隔门对话和紧急救援。

### 日落之城 / Sunset

- 来源：https://valorant.fandom.com/wiki/Sunset
- 地点：Alpha Earth，美国 Los Angeles 的 Boyle Heights。
- 关键机制：传统双点三路结构；B Market 与 Mid Courtyard 间有可开关且可破坏的机械门。
- 剧情背景：Kingdom 在当地有秘密设施，居民早已注意到奇怪生物和巨大声响。随后设施前方突然出现巨型塌陷坑，Kingdom 急于掩盖原因，防止机密项目曝光。
- 场景氛围：洛杉矶街区、涂鸦、黄昏、社区生活、地下设施、塌陷危机。
- 适合剧情：社区保护、秘密实验泄露、Gekko 洛杉矶线、街头追逐、媒体封锁。
- 乙游钩子：适合把“日常街区”写成情感锚点。角色不是只保护世界，也保护一条街、一家店、一个朋友。

### 幽邃地窟 / Abyss

- 来源：https://valorant.fandom.com/wiki/Abyss
- 地点：Alpha Earth，挪威 Jan Mayen。
- 关键机制：无外边界，有死亡坠落区；部分路线需要跳过深渊；有绳索、可破坏门和狭窄外沿捷径。
- 剧情背景：Jan Mayen 北极岛地下藏着 Scions of Hourglass 的 Elder Grotto 要塞。VALORANT Protocol 在与 Hourglass 初次冲突后夺取此地，里面仍残留大量千年级文物、记录和秘密。
- 场景氛围：北极地下堡垒、深渊、古老组织、禁忌文献、坠落恐惧。
- 适合剧情：Hourglass 主线、禁忌档案、叛徒审讯、悬崖救援、失控坠落。
- 乙游钩子：适合高压亲密场景。断崖边的牵手、绳索上的信任、地下文献室里的秘密告白都很自然。

### 盐海矿镇 / Corrode

- 来源：https://valorant.fandom.com/wiki/Corrode
- 地点：Omega Earth，法国 Normandy 的 Mont-Saint-Michel。
- 关键机制：信息框显示无特殊机关，适合强调地形、盐矿和城市结构本身。
- 剧情背景：Omega 的 First Light 后气候剧变使 Mont-Saint-Michel 周边海水蒸发，暴露出巨大盐滩。当地被改造成源晶盐矿设施，居民进行开采和处理。
- 场景氛围：盐海、干涸海床、旧修道院城镇、矿业设施、Omega 气候灾难后的新生计。
- 适合剧情：气候灾后社会、矿工生活、Omega 视角、物资争夺、旧圣地被工业化。
- 乙游钩子：适合写“敌方世界也有人活得很艰难”。主角可以在这里遇到并非恶人的 Omega 平民或守卫。

### 训练场 / Range

- 来源：https://valorant.fandom.com/wiki/Range
- 地点：Alpha Earth，意大利 Venice 附近 Poveglia；设定上是 VALORANT HQ simulation。
- 关键机制：基础训练、能力测试、武器测试、跑酷课程；部分区域模拟 Ascent 的 B 点结构。
- 剧情背景：它更像 VALORANT 的训练与模拟空间，可承载新成员入队、能力测试、武器适配、战术演练和队内日常。
- 场景氛围：基地、晨训、靶场、模拟任务、隐藏办公室、地下音乐和“Top Secret”房间。
- 适合剧情：入队第一天、导师线、训练赛、能力失控、队友观战。
- 乙游钩子：最适合做开局教程章节。所有可攻略角色都可以通过训练风格被快速立住。

## 场景组合建议

- 初遇与入队：Range → Haven → Split。先训练，再用寺院/都市任务展示角色价值。
- 身份误会线：Ascent → Icebox → Fracture。先由威尼斯灾难引出同位体，再查古船和实验设施。
- Omega 观察线：Pearl → Lotus → Corrode。让主角逐步发现 Omega 世界的民生、遗迹和灾后工业。
- 企业压迫线：Bind → Split → Sunset。三张地图都适合 Kingdom 与本地居民冲突。
- Hourglass 主线：Abyss → Vyse/Clove/Iso 相关任务 → Corrode 或 Lotus。适合往超自然组织和古代知识扩展。
